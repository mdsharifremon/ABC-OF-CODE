# VSCode Shortcuts

### Command Platte
<kbd>ctl</kbd>+<kbd>shift</kbd>+<kbd>p</kbd>
### Multicursor
<kbd>alt</kbd> + <kbd>click</kbd>
### Multi word select
<kbd>ctl</kbd>+<kbd>d</kbd><br>
<kbd>ctl</kbd>+<kbd>shift</kbd>+<kbd>alt</kbd>+<kbd>j</kbd>
### Find word
<kbd>ctl</kbd>+<kbd>f</kbd>
### Replace All
<kbd>ctl</kbd>+<kbd>alt</kbd>+<kbd>enter</kbd>
